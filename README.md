# 🐭 鼠鼠偷吃插件 (astrbot_plugin_touchi) 🎁

这是一个为 AstrBot 开发的趣味盲盒插件。插件包含两种主要结果：威龙珍藏美图或触发"鼠鼠偷吃"。

**使用指令 touchi**
插件配置中可关闭美图功能 启用纯享偷吃模式
使用自定义戳一戳配合此插件效果更好哦  预计更新（指令控制cd）（个人图鉴）

安装方法
将插件文件夹 astrbot_plugin_touchi 放入 AstrBot 的 data/plugins/ 目录
